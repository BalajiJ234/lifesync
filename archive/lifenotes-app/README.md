# LifeNotes App - Archived Code

This directory contains the Notes and Todos features extracted from LifeSync for future use as a standalone application.

## Contents

### Pages
- `pages/notes/` - Notes management page with rich text editing
- `pages/todos/` - Todo list with categories, priorities, and due dates

### Store (Redux Slices)
- `store/notesSlice.ts` - Notes state management with CRUD operations
- `store/todosSlice.ts` - Todos state management with filtering and sorting

## Features to Build

### Notes
- Create, edit, delete notes
- Rich text content
- Search functionality
- Categories/tags

### Todos  
- Create, edit, delete todos
- Priority levels (low, medium, high)
- Due dates
- Categories
- Completion tracking
- Filters (all, active, completed)

## To Use This Code

1. Create a new Next.js project
2. Copy these files to appropriate locations
3. Set up Redux store with these slices
4. Add necessary dependencies (redux-toolkit, redux-persist, lucide-react)
5. Update imports as needed
